git branch -r (list branches)

git checkout -b branch-name origin/branch-name (one way to bring down a new branch)
or
git fetch origin branch-name (second way to bring down a new branch)
git checkout branch-name

git branch -d branch-name  (off of branch after merge)
git branch -D branch-name  (off of branch)
git branch -m branch-new-name  (on branch)

git switch branch-name
git switch -c branch-name

git merge branch-name (merges branch-name into branch we are currently on)
git merge --abort
